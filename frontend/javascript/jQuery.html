# 层次选择器

层次选择器通过DOM元素间的层次关系获取元素，其主要的层次关系包括后代、父子、相邻、兄弟关系，通过其中某类关系可以方便快捷地定位元素

选择器 功能描述
ancestor descendant 根据祖先元素匹配所有的后代元素
parent>child 根据父元素匹配所有的子元素
prev+next 匹配prev后的相邻元素
prev-siblings 匹配prev后面的所有兄弟元素

# 过滤选择器 ":"
